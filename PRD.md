# 產品描述

用 python 寫一個視窗程式，輸入 2 個台灣股市的股票代碼後，顯示這些股票和此股票代碼在 120 日、20 日和 60 日股價的相關係數，你可以使用 yfinance 的資料當做股價來源。

## 系統需求

- 資料儲存為 SQLite
- 若使用者輸入的兩檔股票的最近 120 個交易日的收盤價不在 SQLite 中，或最新（前一個交易日）的資料不存在，則到 yfinance 抓取後存入，這個股票資料是累積的
- 捉取資料時，請注意使用者輸入的只有股票代號，沒有.TW 或.TWO 區分的尾碼，你在 yfinance 抓取資料時，要自己加上
- 用 Tkinter 框架建立一個視窗程式
- 使用者按下查詢相關後，顯示這兩檔股票在最近 120、60 和 20 個交易日中，股價的相關係數。
