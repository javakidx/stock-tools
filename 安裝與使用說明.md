# 台股相關性分析系統 - 安裝與使用說明

## 快速開始

### 方法一：使用啟動腳本（推薦）

1. 雙擊 `啟動程式.command` 檔案
2. 程式會自動啟動

### 方法二：使用終端機

1. 開啟終端機（Terminal）
2. 進入程式目錄：
   ```bash
   cd /Users/bioyang/dev/vibe-coding/stock-tools
   ```
3. 啟動虛擬環境：
   ```bash
   source .venv/bin/activate
   ```
4. 執行程式：
   ```bash
   python3 main_gui.py
   ```

## 使用說明

### 1. 輸入股票代碼

- 在「股票代碼 1」欄位輸入第一檔股票代碼（例如：2330）
- 在「股票代碼 2」欄位輸入第二檔股票代碼（例如：2317）

### 2. 計算相關係數

- 點擊深藍色的「計算相關係數」按鈕
- 等待計算完成（通常幾秒鐘）

### 3. 查看結果

程式會顯示：
- 兩檔股票的基本資訊
- 120 日、60 日、20 日的相關係數
- 相關性強度說明

## 相關係數解讀

- **1.0**：完全正相關（兩股票完全同步）
- **0.7~0.9**：強正相關
- **0.4~0.7**：中等正相關
- **0.0~0.4**：弱正相關
- **0.0**：無相關
- **-0.4~0.0**：弱負相關
- **-0.7~-0.4**：中等負相關
- **-0.9~-0.7**：強負相關
- **-1.0**：完全負相關（一漲一跌）

## 常見問題

### Q: 找不到股票資料怎麼辦？

A: 確認：
1. 股票代碼是否正確（台股代碼為 4 位數字）
2. 資料庫中是否有該股票的資料
3. 該股票是否為台灣上市/上櫃股票

### Q: 程式無法啟動？

A: 確認：
1. 已安裝 Python 3.8 以上版本
2. 虛擬環境已正確設定
3. 所有相依套件已安裝（執行 `uv pip install -r requirements.txt`）

### Q: 如何更新股票資料？

A: 目前程式會自動從 yfinance 取得資料並儲存到 SQLite 資料庫中。

## 關於打包成 .app 檔案

由於目前使用 Python 3.14，打包工具（PyInstaller、py2app）對此版本的支援還不完善，特別是 tkinter 模組。

如果您需要製作獨立的 macOS 應用程式檔案（.app），建議：

1. **使用 Python 3.11 或 3.12**：
   ```bash
   # 安裝 Python 3.12
   brew install python@3.12

   # 使用 Python 3.12 重新建立虛擬環境
   python3.12 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt

   # 使用 PyInstaller 打包
   pip install pyinstaller
   pyinstaller stock-tools.spec --clean
   ```

2. **或繼續使用啟動腳本**：
   - 啟動腳本（.command）在 macOS 上同樣方便
   - 雙擊即可執行
   - 無需安裝額外工具
